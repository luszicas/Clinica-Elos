# 🌿 Clínica Elos | Cuidar é criar Elos

Este projeto consiste em uma **Landing Page institucional** para a Clínica Elos, focada em serviços de **psicologia e bem-estar emocional**.

O objetivo foi criar uma interface que transmita **serenidade, credibilidade e acolhimento**, facilitando o acesso do paciente ao suporte profissional através de uma navegação intuitiva e estratégica.

---

## 🌐 Acesse o Projeto

🔗 **Site online:**  
👉 https://clinicaelos.netlify.app/

---

## 🚀 Tecnologias e Implementação

O desenvolvimento foi pensado para ser **leve, acessível e com estética soft-minimalist**.

### 🏗️ Arquitetura

**HTML5 Semântico**  
Estrutura organizada garantindo acessibilidade e melhor leitura para motores de busca (SEO).

### 🎨 Estilização

**CSS3 puro**, com foco em:

- Flexbox e CSS Grid para organização dos cards
- Paleta de cores neutras e tons terrosos
- Layout clean e profissional
- Hierarquia visual clara

### ⚡ Interatividade

**JavaScript Vanilla** para:

- Validação simples do formulário de contato
- Comportamento do menu de navegação
- Scroll suave (smooth scroll)
- Pequenos aprimoramentos de UX

### 📱 Design Responsivo

Layout totalmente adaptável para dispositivos móveis, garantindo que o botão de contato esteja sempre acessível.

---

## 📱 Diferenciais do Projeto

- 🎯 **Foco em Conversão**  
  Estrutura com CTAs estratégicos para agendamentos e contato direto.

- 🤝 **Seção "Quem Somos" Humanizada**  
  Layout com fotos e textos alinhados para fortalecer conexão emocional.

- 🧠 **Cards de Especialidades**  
  Organização clara das áreas de atuação (ansiedade, depressão, luto, etc.).

- 🏢 **Galeria do Ambiente**  
  Espaço dedicado a apresentar o consultório, transmitindo segurança antes da consulta.

- 📩 **Formulário Integrado**  
  Interface pronta para captação de leads e dúvidas.

---

## 🛠️ Estrutura da Página

- **Header** → Menu fixo e minimalista  
- **Hero Section** → Mensagem acolhedora com CTA principal  
- **Sobre a Clínica** → História e missão das profissionais  
- **Especialidades** → Lista detalhada de serviços  
- **Quando buscar ajuda?** → Checklist interativo  
- **Galeria** → Visualização do consultório  
- **Depoimentos** → Prova social para gerar autoridade  
- **Contato** → Formulário + links para redes sociais/WhatsApp
